What is page?

On the World Wide Web, a page is a file notated with the Hypertext Markup Language (HTML). Usually, it contains text and specifications about where image or other multimedia files are to be placed when the page is displayed. You can think of a Web site as a book (albeit a hypertext kind of book rather than a sequentially arranged kind of book) that arrives a page at a time as you request each one.

Each page is an individual HTML file with its own Web address (URL). The first page you usually request at a site is known as the home page. (Most home pages have a default name that doesn't have to be specified; you only need to enter the domain name for the site itself.) With frames, multiple pages (HTML files) can be downloaded to a browser and presented on designated sections of the display screen at the same time.
